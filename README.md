# Landing Page Corporativa - Clube Aldeia da Serra

## 📝 Sobre o Projeto
Desenvolvimento do site oficial *One Page* para o Clube Aldeia da Serra. O objetivo deste projeto foi digitalizar a comunicação do clube, centralizando informações sobre horários, eventos e comunicados para os sócios em uma interface rápida, responsiva e de fácil manutenção. 

Além do desenvolvimento Front-End, o projeto englobou a estruturação da infraestrutura de hospedagem e a negociação de um SLA (Service Level Agreement) de manutenção contínua B2B.

## 🚀 Tecnologias e Ferramentas
* **Front-End:** HTML5, CSS3 e JavaScript (Vanilla).
* **Infraestrutura/Deploy:** Cloudflare (Gestão de DNS, CDN e otimização de performance sem custos de servidor).
* **Design:** Abordagem *Mobile-First* para garantir usabilidade em qualquer dispositivo.

## 💼 Desafios e Soluções (Business Model)
* **Problema:** O clube precisava de uma vitrine digital eficiente, mas com baixo custo operacional.
* **Solução:** Uma arquitetura estática (livre de bancos de dados complexos na fase inicial), hospedada via Cloudflare, garantindo alta disponibilidade.
* **Modelo de Negócio:** Implantação do site e fechamento de um contrato recorrente de manutenção, com fluxo de atualizações semanais bem definido.
